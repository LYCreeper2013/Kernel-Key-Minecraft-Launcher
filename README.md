# KKMCL - Minecraft 启动器

[![Java](https://img.shields.io/badge/Java-17+-blue)](https://adoptium.net/)

一个纯净、轻量、跨平台的 Minecraft 自定义启动器。

- ✅ 支持 Minecraft 正式版 / 快照版 / 旧版本
- ✅ 支持 Forge / Fabric  等模组加载器
- ✅ 多实例管理
- ✅ 自动安装与修复

---

## 下载与使用

👉 前往 [Releases](https://github.com/LYCreeper2013/Kernel-Key-Minecraft-Launcher-Version) 下载对应系统的启动器包。

| 系统 | 格式              |
|----|-----------------|
| Windows | `.jar` 或 `.zip` |
| macOS | `.jar`          |
| Linux | `.tar.gz`       |

> 如使用源码运行，请确保已安装 Java 21+。
